## Features

<CardGrid stagger>
	<Card title="Read Operations" icon="magnifier">
		Query branches, commits, pull requests, diffstats, pipelines, and repositories across a workspace.
	</Card>
	<Card title="Write Operations" icon="pencil">
		Create, merge, approve, and decline pull requests. Create, resolve, and delete PR comments.
	</Card>
	<Card title="Cross Platform" icon="laptop">
		Pre-compiled binaries for macOS, Linux, and Windows ready to run via Claude Desktop or Cursor.
	</Card>
	<Card title="Secure Auth" icon="seti:lock">
		Supports seamless OAuth 2.0 flow or declarative API token setups for CI pipelines.
	</Card>
